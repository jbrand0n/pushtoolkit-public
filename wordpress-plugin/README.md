# PushToolkit WordPress Plugin

This directory contains the official WordPress plugin for PushToolkit, enabling seamless integration between WordPress and the PushToolkit push notification platform.

## What's Inside

- **pushtoolkit-wp/**: The complete WordPress plugin

## Quick Start

1. **Install the plugin**:
   ```bash
   cp -r pushtoolkit-wp /path/to/wordpress/wp-content/plugins/
   ```

2. **Activate it** in WordPress admin under Plugins

3. **Configure** the plugin with your PushToolkit API credentials

See the [plugin README](pushtoolkit-wp/README.md) for detailed installation and usage instructions.

## Features

- Automatic push notifications when publishing posts
- Manual notification sending from WordPress dashboard
- Subscriber management interface
- Analytics and performance tracking
- Per-post notification customization
- Service worker auto-configuration

## Requirements

- WordPress 5.0+
- PHP 7.4+
- HTTPS-enabled website
- Running PushToolkit instance
